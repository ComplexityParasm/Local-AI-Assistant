# Local-AI-Assistant
Qwen Local AI Assistant — учебный проект по дисциплине, связанной с искусственным интеллектом.   Основная цель проекта — показать, как современную языковую модель можно развернуть локально на персональном компьютере и использовать для разработки, автоматизации и генерации контента.
